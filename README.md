# Projet réalisé

**Dans ce répertoire vous trouverez les différents projets réalisés que sa soit des mini-projet ou des projets avancés**.

Il y aura deux répertoires un pour les mini projets et l'autre pour les projets avancés :

- mini-projet : des parties de code qui concernent une éxécution donnée par exemple ingestion de données dans une base de données postegre et requêtage dans postegre admin.
- Projet avancés : Un projet menés de bout-en-bout de l'extraction en passant par l'ingestion et l'orchestration, la phase de préparation voir la phase d'analyse ( pas très pousséés).

**Partez à la découverte de mon travail, espérant que sa vous plaise.**
